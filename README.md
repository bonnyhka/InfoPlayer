# InfoPlayer

Панель статистики игроков для Minecraft 1.21.1 и NeoForge от 21.1.227 до 21.1.233.

## Возможности

- Команда `/infoplayer` доступна всем игрокам.
- Список игроков в сети и игроков, которые ранее заходили на сервер.
- Скин, имя, статус, уровень, общий опыт, наигранное время и прогресс достижений.
- Подробная карточка со здоровьем, голодом, режимом игры, измерением, координатами и датами активности.
- Просмотр полного инвентаря, хотбара, брони и левой руки.
- Необязательная интеграция с Curios: отдельная вкладка аксессуаров, поддержка обычных и косметических слотов.
- Полноценная модель игрока показывает надетую броню и предметы в обеих руках.
- Для офлайн-игроков показывается последний сохранённый снимок инвентаря.
- Администраторы с уровнем прав 2 или выше могут забирать предметы у игроков в сети.
- Администраторы могут забирать и аксессуары Curios, если сам аксессуар разрешает снятие.
- Каждый игрок может отдельно скрыть от других статус, активность, опыт, достижения, время игры, здоровье, голод, режим, измерение, координаты, дату первого входа, инвентарь и Curios.
- Изменения приватности применяются в реальном времени к открытому списку и карточке игрока.
- Администраторы всегда видят все данные, а владелец профиля сохраняет полный доступ к себе.
- Полностью кастомный зелёный интерфейс и собственные кнопки.
- Все подписи интерфейса на русском языке.
- Каждое изменение инвентаря повторно проверяет права на сервере.
- Данные сохраняются в `<папка мира>/infoplayer_players.json`.

## Установка

Один и тот же JAR-файл должен быть установлен на сервер и на клиенты игроков. После перезапуска сервера используйте:

```text
/infoplayer
```

Поддерживаются версии NeoForge `21.1.227`, `21.1.228`, `21.1.229`, `21.1.230`, `21.1.231`, `21.1.232` и `21.1.233`.

Curios не является обязательным: без него мод работает как обычно. Интеграция проверена с Curios `9.5.1+1.21.1`.

## Сборка

Требуется Java 21.

```powershell
.\gradlew.bat build
```

Готовый файл появится в `build/libs/infoplayer-1.6.0.jar`.

## Важно

История игроков начинает собираться после установки мода. Точное время последнего входа до установки восстановить невозможно.
